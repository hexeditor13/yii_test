# Раздел “Накладные”
На странице таблица. Колонки: Откуда, Куда, Получатель, Статус. По каждому столбцу
Реализована сортировка.
Возможность создания накладной, редактирования, удаления.
Создание и редактирование накладных происходит в модальных окнах, без
перезагрузки страницы.
Есть файл с видеодемонстрацией.
